# Hola, soy Manuel de Jesús Ramírez Morales 👋

**Desarrollador Full-Stack | Enfocado en Migraciones Web y Automatización**

Soy un desarrollador de software con 2 años de experiencia en el diseño, mantenimiento y modernización de sistemas empresariales. Me gusta resolver problemas reales y actualmente me enfoco en la transición de arquitecturas *legacy* hacia entornos web más modernos, así como en la integración de APIs y el despliegue de servidores.

## 🚀 Lo que estoy haciendo ahora

* **Migración de ERP:** Actualmente trabajo en la migración integral de un sistema de planificación de recursos empresariales (ERP) pasando de **GeneXus** a **Laravel**.
* **Reingeniería y Optimización:** Traduciendo la lógica de negocio, refactorizando el código y mejorando los flujos de trabajo para lograr un sistema web ágil y eficiente.

## 🛠️ Experiencia y Enfoque

* **Integración de Servicios (APIs):** Conexión e implementación de facturación electrónica (Timbrado Profact), plataformas de mensajería (SMSMasivos) y automatización de correos electrónicos.
* **Administración de Servidores:** Despliegue, configuración y mantenimiento básico de entornos de desarrollo y producción sobre distribuciones **Linux** (CentOS, openSUSE Leap, Fedora, Nobara).
* **Procesamiento de Datos:** Manejo de estructuras XML, extracción automatizada de datos (utilizando herramientas como pytesseract y pdf2image) y armado de consultas SQL para auditorías o análisis.
